# MuchaCurve
～ みゅしゃかーぶ ～

「YMM4で細かい色調補正がしたいな...」と思うことがあったので、自分用に制作したものです。<br>
こちらは、**YMM4用の映像エフェクトプラグイン**となっています。<br>
DaVinci Resolveに準拠した7種類のカーブを追加し、ポイントを追加してカーブの形を変えることで、高度な色調補正を行えます。

---

## 💡 使い方

1. 映像エフェクト欄から、**「みゅしゃかーぶ」**を探して追加します。
2. 7つのカーブが表示されます。
   - `Custom Curves` / `Hue vs Hue` / `Hue vs Sat` / `Hue vs Luma` / `Luma vs Sat` / `Sat vs Sat` / `Sat vs Luma`
3. お好みでポイントを追加してカーブの形を変え、色調補正をしてください。

---

## 🛠️ カーブの操作方法

* **ポイントの追加**
  * カーブの近くを**シングルクリック**すると、新たにポイントが追加されます。
* **ポイントの種類の変更**
  * 既存のポイントをクリックすると、ポイントの種類を三種類から変更できます。
    * **◯** ： 自由に動かせる、ベジェ曲線のアンカーポイント
    * **◇** ： 自由に動かせる、角ばった頂点のポイント
    * **●** ： 横方向にのみ動かせる、ベジェ曲線のアンカーポイント
* **スポイト機能**
  * 一部のカーブには、スポイトボタンが実装されています。<br>スポイトボタンを押して適当な色を選ぶと、横軸において選んだ色と同じ座標に自動でポイントが追加されます。
* **ポイントの削除・統合**
  * ポイントを**ダブルクリック**すると、ポイントを削除できます。<br>また、ポイント同士の距離が非常に近い場合、自動で一つのポイントに統合されます。
* **リセット**
  * **リセットボタン**を押すと、カーブの形が初期状態に戻ります。

## 📊 グラフ紹介

本プラグインは「カスタムカーブ」と「6つのセカンダリカーブ」で構成されています。

### 1. Custom Curves（カスタムカーブ）
最も基本的なカーブで、画面全体の明るさ（Y）や、赤・緑・青（RGB）の各チャンネルを調整できます。<br>
コントラストの調整や、特定の色の被り（カラーキャスト）の補正に使えます。

### 2. Hue vs Hue（色相 vs 色相）
**「ある特定の色」を「別の色」に変化**させます。<br>
特定の被写体の色味を微調整したいときや、色の置き換えに使えます。

### 3. Hue vs Sat（色相 vs 彩度）
**「特定の色」の「鮮やかさ（強さ）」を調整**します。<br>
「肌の色はそのままで、背景の緑だけを鮮やかにする」のような、色の強調や分離に使えます。

### 4. Hue vs Luma（色相 vs 輝度）
**「特定の色」の「明るさ」を調整**します。<br>
青空を暗くして偏光フィルターみたいな感じにしてみたり、肌の色を少し明るくして透明感を出したりできます。

### 5. Luma vs Sat（輝度 vs 彩度）
**「明るさの範囲」を指定して、その部分の「彩度」を調整**します。<br>
暗い部分に乗った変な色味（ノイズっぽさ）を消したり、非常に明るい部分が不自然に色づいたりしたのを消したりできます。

### 6. Sat vs Sat（彩度 vs 彩度）
**「彩度の強さ」に基づいて、さらに「彩度」を調整**します。<br>
画面全体に彩度をかけると「派手になりすぎる場所」が出てしまう場合に、彩度が低い部分だけを底上げしてバランスをとったりできます。

### 7. Sat vs Luma（彩度 vs 輝度）
**「彩度の強さ」に基づいて、その部分の「明るさ」を調整**します。<br>
「色が濃すぎて白飛び（色飛び）している箇所」の輝度を下げ、ディテールを復元したりできます。
